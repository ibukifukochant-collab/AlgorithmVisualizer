## 1. 项目结构与文件创建

创建新的插件目录和文件，遵循现有项目结构：

```
plugins/
└── MazeDFS/
    ├── index.html       # 主页面结构
    ├── style.css        # 插件样式
    ├── algorithm.js     # DFS算法实现
    └── visualization.js # 可视化逻辑
```

## 2. 核心功能实现

### 2.1 算法实现 (algorithm.js)
- 创建 `MazeDFS` 类，实现深度优先搜索算法
- 生成随机迷宫或接受自定义迷宫
- 记录每一步的搜索状态，包括：
  - 当前位置
  - 已访问路径
  - 堆栈状态
  - 操作类型（push/pop）
- 支持起点(0,0)标记为S，终点(n-1,m-1)标记为E

### 2.2 可视化实现 (visualization.js)
- 创建 `MazeDFSVisualizer` 类，继承现有可视化器架构
- 迷宫绘制功能：
  - 清晰的x轴和y轴坐标
  - 墙壁、路径、当前位置的不同颜色
  - 起点S和终点E的特殊标记
- 堆栈可视化：
  - 右侧实时显示堆栈变化
  - 绿色动画表示压栈操作
  - 红色闪烁表示弹栈操作
  - 堆栈接近满时的滚动效果
  - 弹栈时的反向滚动效果
- 动画控制：
  - 播放/暂停
  - 上一步/下一步
  - 速度调节
  - 重置功能

### 2.3 页面结构 (index.html)
- 采用与BubbleSort插件相同的页面结构
- 包含：
  - 算法描述和复杂度信息
  - 迷宫参数设置（宽度、高度、墙壁密度）
  - 可视化画布
  - 堆栈显示区域
  - 控制面板

### 2.4 样式设计 (style.css)
- 遵循项目现有的UI风格
- 响应式设计
- 迷宫网格样式
- 堆栈区域样式
- 动画效果样式
