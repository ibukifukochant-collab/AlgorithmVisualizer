## 插件结构

每个插件都应该包含以下文件：

```
plugins/
└── PluginName/
    ├── manifest.json          # 插件元数据
    ├── algorithm.js           # 算法实现
    ├── visualization.js       # 可视化实现
    ├── style.css             # 插件样式
    ├── index.html            # 插件入口页面
    └── thumbnail.png         # 插件缩略图
```

### 文件说明

| 文件 | 用途 | 必需 |
|------|------|------|
| `manifest.json` | 插件元数据配置 | ✅ |
| `algorithm.js` | 算法核心实现 | ✅ |
| `visualization.js` | 可视化逻辑 | ✅ |
| `style.css` | 插件样式 | ✅ |
| `index.html` | 插件界面 | ✅ |
| `thumbnail.png` | 插件缩略图 | ✅ |

## 开发规范

### 命名规范
- **目录名**: 使用PascalCase，如 `BubbleSort`, `MazeDFS`
- **文件名**: 使用kebab-case，如 `algorithm.js`, `style.css`
- **类名**: 使用PascalCase，如 `BubbleSortAlgorithm`, `MazeDFSVisualizer`
- **变量名**: 使用camelCase，如 `currentStep`, `animationSpeed`

### 目录结构要求
```
plugins/
└── BubbleSort/
    ├── manifest.json
    ├── algorithm.js
    ├── visualization.js
    ├── style.css
    ├── index.html
    └── bubble-sort.png
```

## 代码规范

### Algorithm.js 规范
```javascript
// 算法实现类模板
class AlgorithmName {
    constructor() {
        this.steps = [];
        this.currentStep = 0;
    }

    // 初始化算法
    initialize(params) {
        this.steps = [];
        this.currentStep = 0;
        
        // 记录初始状态
        this.recordStep({
            type: 'initialize',
            data: params,
            description: '算法初始化完成'
        });
        
        return this.steps;
    }

    // 执行算法
    execute() {
        // 算法逻辑实现
        // 每一步都要记录状态
        this.recordStep({
            type: 'operation',
            data: { /* 当前状态数据 */ },
            description: '执行操作描述'
        });
        
        return this.steps;
    }

    // 记录步骤
    recordStep(step) {
        step.stepNumber = this.steps.length;
        this.steps.push(step);
    }

    // 获取当前步骤
    getCurrentStep() {
        return this.steps[this.currentStep];
    }

    // 下一步
    nextStep() {
        if (this.currentStep < this.steps.length - 1) {
            this.currentStep++;
            return this.getCurrentStep();
        }
        return null;
    }

    // 上一步
    prevStep() {
        if (this.currentStep > 0) {
            this.currentStep--;
            return this.getCurrentStep();
        }
        return null;
    }
}

// 导出算法类
if (typeof module !== 'undefined' && module.exports) {
    module.exports = AlgorithmName;
}
```

### Visualization.js 规范
```javascript
// 可视化类模板
class AlgorithmNameVisualizer {
    constructor(canvasId, infoDisplayId) {
        this.canvas = document.getElementById(canvasId);
        this.ctx = this.canvas.getContext('2d');
        this.infoDisplay = document.getElementById(infoDisplayId);
        this.steps = [];
        this.currentStep = 0;
        this.animationSpeed = 500;
        this.isPlaying = false;
        
        this.initCanvas();
    }

    // 初始化画布
    initCanvas() {
        this.canvas.width = 600;
        this.canvas.height = 400;
        this.ctx.fillStyle = '#667eea';
        this.ctx.strokeStyle = '#fff';
        this.ctx.lineWidth = 2;
    }

    // 加载数据
    loadData(data) {
        this.data = data;
        this.draw();
    }

    // 加载步骤
    loadSteps(steps) {
        this.steps = steps;
        this.currentStep = 0;
        this.draw();
    }

    // 绘制当前状态
    draw() {
        this.clearCanvas();
        
        if (this.steps.length === 0) {
            this.drawInitialState();
            return;
        }
        
        const step = this.steps[this.currentStep];
        this.updateInfoDisplay(step);
        this.drawStep(step);
    }

    // 更新信息显示
    updateInfoDisplay(step) {
        this.infoDisplay.innerHTML = `
            <div class="step-description">${step.description}</div>
            <div class="step-counter">
                <span>步骤: ${step.stepNumber + 1}/${this.steps.length}</span>
            </div>
        `;
    }

    // 设置动画速度
    setSpeed(speed) {
        this.animationSpeed = speed;
    }

    // 播放/暂停
    playPause() {
        this.isPlaying = !this.isPlaying;
        if (this.isPlaying) {
            this.playAnimation();
        }
    }

    // 播放动画
    playAnimation() {
        if (this.isPlaying && this.currentStep < this.steps.length - 1) {
            setTimeout(() => {
                this.nextStep();
                this.playAnimation();
            }, this.animationSpeed);
        } else {
            this.isPlaying = false;
        }
    }

    // 下一步
    nextStep() {
        if (this.currentStep < this.steps.length - 1) {
            this.currentStep++;
            this.draw();
            return true;
        }
        return false;
    }

    // 上一步
    prevStep() {
        if (this.currentStep > 0) {
            this.currentStep--;
            this.draw();
            return true;
        }
        return false;
    }

    // 重置
    reset() {
        this.currentStep = 0;
        this.isPlaying = false;
        this.draw();
    }

    // 清空画布
    clearCanvas() {
        this.ctx.clearRect(0, 0, this.canvas.width, this.canvas.height);
    }
}
```

## 常见问题

### Q: 插件没有在主页面显示？
A: 检查以下事项：
- manifest.json 格式是否正确
- 插件目录是否在 plugins/ 下
- 服务器是否重启以加载新插件

### Q: 样式不生效？
A: 检查：
- CSS文件路径是否正确
- 类名是否与HTML对应
- 是否有CSS语法错误

### Q: 算法步骤不显示？
A: 检查：
- algorithm.js 是否正确记录每一步
- visualization.js 是否正确加载步骤数据
- 控制逻辑是否正确连接

### Q: 如何调试插件？
A: 使用浏览器开发者工具：
- 检查Console标签查看错误信息
- 使用Network标签检查资源加载
- 使用Elements标签检查DOM结构
